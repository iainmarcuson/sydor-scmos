Sydor GenCam Code Generation
=============================

Introduction
------------

The readback parameters and code are programatically generated.  A template and two utilities are provided to generate the database and readback code.

Database
--------

1. `python3 gendb.py query_template.txt query.db`
2. `cp query.db ../Db`

Code
----

1. `python3 gen_query_code.py query_template.txt query_code.c`
